\section{Methodology}

This report was automatically generated using the \texttt{latex-report-generator} skill for Claude Code.

\subsection{Data Collection}

Content was collected using the \texttt{web-scraper} skill, which:
\begin{itemize}
    \item Respects robots.txt directives
    \item Enforces rate limiting (2-second minimum between requests)
    \item Validates domains against scientific whitelist
    \item Logs all activity for transparency
\end{itemize}

\subsection{Figure Selection}

Figures were auto-selected using a scoring algorithm that evaluates:
\begin{itemize}
    \item Image size and dimensions (prefer high-resolution images)
    \item Alt text quality (prefer descriptive captions)
    \item Filename descriptiveness
    \item Aspect ratio and format
\end{itemize}

The top-ranked figures were selected for inclusion in this report.
